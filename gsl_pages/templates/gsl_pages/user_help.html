Foire aux questions

Si on accepte un projet sur Turgot, le statut Démarche numérique reste "En instruction".

L'acceptation sur Turgot coche la dotation acceptée puis met à jour les annotations liées à cette dotation (montant, taux, assiette) sur Démarche numérique.

Pour passer le projet en statut "Accepté" sur Démarche numérique, il faut cliquer sur notifier le projet dans la section "Notifications" de la page projet sur Turgot.

Depuis l'onglet "Notifications" de la page projet, suivez ces étapes :

  1. Générez un arrêté à partir d'un modèle d'arrêté.
  2. Générez une lettre de notification à partir d'un modèle de lettre de notification.
  3. Faites signer ces deux documents, puis réimportez-les dans un seul fichier au format pdf (lettre et arrêté réunis).
  4. Cliquez sur "Rédiger le message de notification" pour écrire le message à envoyer au demandeur. Vous pouvez y ajouter une motivation (facultatif) et des annexes. Le fichier signé sera automatiquement joint.
  5. Cliquez sur "Confirmer l'envoi" : le projet passera alors au statut "Accepté" sur Démarche numérique.

Oui, il existe une synchronisation entre les annotations sur Démarche numérique et les informations affichées sur Turgot.

Les champs "cases à cocher" suivants sont toujours synchronisés, quel que soit le statut du projet sur Turgot :

  • QPV, FRR
  • Contractualisation
  • Zonage
  • Budget vert

Il convient de distinguer entre :

  • L'assiette du projet, qui est toujours récupérée depuis Démarche numérique, indépendamment du statut du projet sur Turgot ;
  • Les montants et taux renseignés dans les annotations privées de Démarche numérique, qui sont quant à eux synchronisés uniquement si le projet est accepté sur Turgot.

Cela dépend des annotations renseignées sur le dossier Démarche numérique au moment de l'acceptation.

Cas 1 — Aucune dotation cochée

Turgot n'est pas mis à jour. Le projet reste dans son état actuel sur Turgot, qu'il soit "En traitement" ou dans un autre statut.

Cas 2 — Des dotations sont cochées, mais aucun montant n'est renseigné

Turgot passe le projet à "Accepté à 0 €". Seules les dotations cochées sont conservées. Si aucune assiette n'est renseignée sur Démarche numérique, l'assiette est mise à 0 €.

Cas 3 — Des dotations sont cochées et un montant est renseigné

Turgot passe le projet à "Accepté". Seules les dotations cochées sont conservées, et le montant renseigné sur Démarche numérique leur est appliqué.

L'assiette est mise à jour dans les deux sens selon ces règles :

Démarche Numérique → Turgot

Si vous cochez une dotation sur Démarche Numérique et renseignez son assiette, Turgot la récupérera automatiquement à la prochaine mise à jour.

Turgot → Démarche Numérique

Si vous acceptez une dotation pour un projet sur Turgot, Turgot va cocher la dotation sur Démarche Numérique et mettre à jour l'assiette, le montant et le taux dans les annotations Démarche Numérique.
La synchronisation de l'assiette entre Turgot et Démarche Numérique sera alors activée.

Turgot vous demandera une motivation et, facultativement, une pièce jointe. Le statut du projet sera ensuite mis à jour sur Turgot et sur DN.

Cas particulier des projets avec double dotation

Si le projet est rattaché à une autre dotation dont le statut n'est pas encore définitif, la notification sur Démarche numérique ne sera pas envoyée immédiatement.

  • Si l'autre dotation est acceptée, le processus de notification se poursuit normalement (génération d'arrêté, notification…) et le dossier sera finalement accepté sur Démarche numérique.
  • Si l'autre dotation est également classée sans suite, alors le dossier Démarche numérique sera lui aussi classé sans suite. En effet, pour un projet en double dotation, si l'une est classée sans suite et l'autre est refusée, alors il va être classé sans suite sur Démarche numérique.

À la prochaine mise à jour, Turgot va automatiquement classer sans suite ou refuser le projet, pour toutes ses dotations s'il en a plusieurs.

Le projet apparaîtra ensuite dans la programmation.

Le retour en instruction est uniquement possible pour les projets qui n'ont pas encore été notifiés au demandeur (typiquement un projet accepté mais non notifié). Dans ce cas, vous pouvez le repasser en traitement depuis la page du projet ou depuis la liste de projets d'une simulation.

Lorsqu'une dotation acceptée repasse en traitement sur Turgot, Turgot décoche automatiquement cette dotation dans les annotations du dossier sur Démarche numérique.

Pour un projet déjà notifié, le retour en instruction doit être initié directement sur Démarche numérique. Démarche numérique se chargera ensuite de mettre à jour Turgot avec le nouveau statut (par exemple "En traitement").

L'export est disponible uniquement sur les simulations. Depuis la page d'une simulation, cliquez sur "Exporter le tableau" pour télécharger la liste des projets dans le format de votre choix (.xls, .xlsx, .csv ou .ods).

Seules les colonnes actuellement affichées dans le tableau sont incluses dans l'export. Vous pouvez personnaliser les colonnes visibles via le bouton "Colonnes affichées" en haut à droite du tableau.

Il y a deux manières de mettre à jour un dossier sur Turgot :
  • les dossiers Démarche numérique sont mis à jour automatiquement toutes les heures piles de 7h à 19h inclus du lundi au vendredi
  • les dossiers Démarche numérique peuvent être mis à jour manuellement en allant sur la page projet et en cliquant sur "Actualiser depuis DN"
